Annie主题使用说明
发布于 2018-08-27   更新于 2019-04-23 说明 热度 0 喜欢 0

一、LOGO和FAVICON

编辑主题目录下的_config.yml文件，设置favicon和avtor（logo）的路径即可。当然，你可使用文字logo或图片logo。

二、页头背景图片

页头背景可以使用随机图片或固定图片，原则上，使用固定图片时，将参数mode设为normal，然后设置normal_url的路径；使用随机图片时，将参数mode设为random，然后设置random_url的路径。

注意跨域问题，尽量不使用跨域的图片url
  • 使用固定图片，请确保参数mode设为normal，然后设置参数normal_url的值
  • 使用unplash随机图片，请确保参数mode设为normal，然后设置参数normal_url的值
  • 使用自定义随机图片，请确保参数mode设为random，然后设置参数random_max、random_url的值

第一步、fork随机图片集 ；第二步、开启你所fork的仓库的page服务；第三步、获取相对地址；第四步、设置random_url为获得的相对地址。

你可以将随机图片集上传到图床（腾讯云、七牛云等），然后获取https://....../Random-img/格式的地址，最后将random_url的值设置为该地址。

你也可以在主题目录下的img文件夹中新建Random-img文件夹，将0.jpg、1.jpg、2.jpg、3.jpg……110.jpg命名格式的图片放入其中，然后将random_url的值设置为/img/Random-img/。

图片的命名使用连续的整数，random_max的值小于等于（0、1、2、3.....110）的最大整数。（ps：如你偏爱访问速度🙃，尽可能使用固定图片）

四、首页的使用

首页文章展示有两种模式：图文模式cart和纯净模式pure。默认图文模式（index-style: cart），你只需编辑主题目录下的_config.yml，设置index-style的参数，即可使用或扩展相关的文章展示模式。

4.1文章封面图

cover参数值即为图片路径，路径可为相对路径或绝对路径。

4.2首页的分页

hexo博客默认首页分页，置参数index_generator.per_page（站点目录下_config.yml）为1、2、3…..以达到分页目的，0则不分页。

4.2文字的截断

excerpt_cart为图文模式的截取字数，推荐小于等于100。excerpt_pure为纯净模式的截取字数，推荐字数为300。

五、归档页说明

Annie主题的归档页按年归档，不分页。当然，可以置参数archive_generator.per_page（站点目录下_config.yml）为1、2、3…..以达到分页目的。

Annie主题中无效，分页是对该页面[主页、归档页、标签页、分类页]的文章进行分页，即每次展示指定数量的文章

11.1代码块复制

只有theme.code_copy.enable为真且config.highlight.enable为假时，代码块的复制功能才有效。

🔻如要显示代码行号，请将line_number设为true，并指定相应代码块的语言类型（html、js、c++、java……）。请务必安装hexo-prism-plugin插件，并选择合适的代码块主题（okaidia，tomorrow，atom-dark……），以使复制模块和代码块的组合效果较为和谐。（推荐tomorrow主题）

十二、评论的功能

Hexo-theme-Annie主题集成了gitalk、DesertsP版的valine、livere评论插件，你可以参考下列文档使用。

  • Gitalk：正式文档、hexo添加gitalk评论插件、Annie主题使用Gitalk
  • Valine：正式文档、加个Valine评论系统
  • Livere：为Hexo博客添加LiveRe评论系统
对于gitalk，你必须设置clientID、clientSecret；对于Valine，你必须设置appid、appkey；对于livere，你必须设置livere_uid。

十三、文章页问题

13.1分类和标签

分类页的HEXO博客字段来源于文章Front-matter中的categories参数值，标签页的说明字段来源于文章Front-matter中的tags参数值。hexo博客框架只支持一个同级分类，支持多个同级标签。

13.2相关的文章

你可以决定是否使用该模块，posts_limit为显示的文章篇数，posts_excerpt为每篇文章的文字截取数量。

只有全站博客文章篇数大于1且`relate.enable`为真时，该模块有效。

13.3阅读量计数

对于leancloud_count，你必须设置appid、appkey；对于busuanzi_count，你只需要开启它即可。

如果你使用过Valine评论插件，那么leancloud_count的使用十分简单。如果你没有使用过，请参考文章Hexo博客next主题添加文章阅读量统计功能创建leancloud应用，并获取其appid、appkey，且设置上面的appid、appkey为相应的值，最后开启visit_post。

13.4文章点赞

如果你完成了13.3阅读量计数的设置，那么只需要开启like_post。

13.5文章排行

如果你完成了13.3阅读量计数的设置，那么只需要开启topN_post。（文章阅读排行依赖于13.3阅读量计数，该模块放置于搜索页面，你可按需定制）

13.6卜蒜子计数

该模块放置于页脚，文章页面只显示文章的阅读量，其他页面只显示总访问量和访客数（等待完善）

13.7数学公式

请先设置主题目录下_config.yml文件的mathjax参数，再于文章的Front-matter中添加mathjax: true

十四、其他的问题

对于social，设置相应的选项即可(你可按需扩展)。对于copyright ，since为站点建立年份，cotent_author为站点作者。对于love，若设置为true，则点击页面时，会出现随机颜色的桃心💚💛💕。

十五、开发定制问题

15.1主题字体问题

主题整体字体，修改body标签的font-family属性

文章详情页字体，修改article-content类的font-family属性

你可以检索css文件，查询相应的标签或类，进而设置相应元素的字体。

15.2页头菜单颜色

页头菜单颜色会根据背景的主题色变化非黑即白。亦即主题色为亮色时，字体为黑；主题色为暗色时，字体为白。你可以参考main.js中函数Annie_ColorExtraction(img)。（等待完善）
